## Overview

emailval is a high-performance email validation library that uses Rust for the heavy lifting while providing a Pythonic API. It achieves 100-500x speedup over pure Python alternatives.

## Key Features

- RFC 5322/6531 compliant validation
- Internationalized domain names (IDN)
- SMTPUTF8 support
- Zero runtime dependencies
- Cross-platform wheels
